##3. 五毒刷题法：

第一遍:

    5分钟：读题 + 思考  
    直接看解法：注意！多解法，比较解法优劣 
    背诵、默写好的解法 -- 很重要！！！
第二遍:

    马上自己写 —> LeetCode 提交 通过
    多种解法比较、体会 —> 优化！
第三遍:

    过了一天后，再重复做题 
    不同解法的熟练程度 —> 专项练习
第四遍:

    过了一周：反复回来练习相同题目
第五遍:

    Interview前一周恢复性训练
